# UI/UX

This document provides a reference to the comprehensive UI/UX guidelines for the Cue d'Etat project.

All visual and interactive design specifications are located in the **[03 UI/UX Guide](./03_UI_UX_Guide/INDEX.md)** directory.

This includes, but is not limited to:

*   **[Gesture Interaction Model](./03_UI_UX_Guide/01_Gesture_Interaction_Model.md):** The definitive guide to all touch gestures (pan, pinch, zoom).
*   **[Visual Style & Theming](./03_UI_UX_Guide/03_Visual_Style_And_Theming.md):** Color palettes, typography, and Material 3 theme implementation.
*   **[Component Specifications](./03_UI_UX_Guide/INDEX.md):** Detailed design and behavior specifications for buttons, sliders, dialogs, and other UI components.
